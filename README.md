# SpudyTube

SpudyTube is a lightweight, single-page web application for browsing and reading premium entertainment news content.

## Overview

The application provides a polished front-end experience with:

- Category-based news browsing
- Search and filtering
- Account creation and login
- Likes, subscriptions, and comments
- Theme preferences (light/dark)
- Local persistence through IndexedDB

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- IndexedDB (client-side storage)

## Getting Started

Because this project is fully static, you can run it directly in a browser.

1. Clone the repository.
2. Open `index.html` in your browser.

For the best experience, you can also serve it with a local static server.

## Project Structure

- `index.html` — Main application (layout, styles, and scripts)
- `README.md` — Project documentation

## Notes

This project currently stores all data in the browser via IndexedDB. User accounts, reactions, and comments are local to the browser profile/device.
